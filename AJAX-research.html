-- Reading Questions

What is AJAX and what is it used for?
AJAX uses xmlHttpRequest object to communicate with seriser. It sends and recieves information.Allows webpages to be updated asynchronously without refreshing the page.
What is JSON and what is it used for?
JavaScript Object Notation. Used for transmitting data in web applications
Can methods be included in a JSON file?
No. it is a string with a specified data format. Only contains properties, no methods.
What does 'jqXhr' represent in the curriculum AJAX examples?
Represents the Ajax request
What's the difference between $.ajax() and $.get()?
$.ajax() gives full control over AJAX request used when other methods do not fullfill your needs. $.get() executes an AJAX GET request. returned data will be passed to the callback handler
Is jQuery required to make AJAX requests?
